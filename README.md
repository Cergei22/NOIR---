# 🍽️ Restaurant - Сайт ресторана

## 📍 URL
`/site/restaurant`

## 📄 Страницы

| Страница | URL | Описание |
|----------|-----|----------|
| Главная (одностраничный) | `/site/restaurant` | Все секции на одной странице |

## 🎨 Секции сайта
- **Hero** - Атмосферный главный экран
- **О ресторане** - История и философия
- **Меню** - Категории блюд с ценами
- **Интерьер** - Фото залов
- **Бронирование** - Форма резервации столиков
- **Контакты** - Адрес и часы работы

## 📸 Изображения
Расположены в `/public/images/restaurant/`:
- `hero.jpg` - главное изображение
- `main.jpg` - основной зал
- `kitchen.jpg` - кухня
- `lounge.jpg` - лаундж зона
- `private.jpg` - приватный зал
- `wine.jpg` - винная карта

## 🔧 Настройка
1. Замените фото интерьера в `/public/images/restaurant/`
2. Обновите меню с актуальными блюдами и ценами
3. Настройте форму бронирования

## 🛠️ Технологии
- Next.js 16 App Router
- Tailwind CSS 4
- Framer Motion
- Lucide Icons
